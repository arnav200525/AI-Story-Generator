**AI Story Generator**
AI Story Generator is a Flask web application that creates customized stories for kids using Google's Generative AI (Gemma model). Users input a main character, story setting, theme, and target age to generate concise, age-appropriate stories under 400 words.
​

Features
- Interactive web interface with character input, dropdown settings (Enchanted Forest, Moon Colony, etc.), theme selection (Adventure, Horror, etc.), and age specification.
- Generates personalized stories via Google Generative AI, ensuring theme consistency and central character focus.​
- Responsive design for smooth user experience; displays stories in a dedicated "Story Theater" page with back navigation.
​
Tech Stack
Backend: Flask (Python)
AI: Google Generative AI (Gemma-3-27b-it model)
Frontend: HTML, CSS, jQuery
Deployment: Local Flask server (debug mode enabled)​

Quick Setup
Clone the repository and navigate to the project directory.
Install dependencies: pip install flask google-generativeai
Add your Google AI API key in app.py (replace ____PASTE__YOUR__API__KEY____).
Run the app: python app.py
